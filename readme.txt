Powerful Team Member block for WordPress with beautiful layouts, social links, and CTA buttons to showcase your team professionally.

== Description ==

WBD Team Member is a modern Gutenberg block plugin that lets you create stunning team member showcases without any design or coding skills. Built for agencies, startups, corporate sites, and freelancers who want a premium team section in minutes.

**Free Features:**

* 12 beautiful pre-designed layouts (Corporate, Agency, Creative Hover, Minimal Card, Centered Profile, Left Aligned, Dark Mode, Gradient Modern, Border Hover, Image Overlay, Split Layout, Compact Grid)
* Add unlimited team members per block
* Profile image with circle, square, or rounded shape options
* Name, designation, short bio, and social links for each member
* Social links for Facebook, LinkedIn, Twitter/X, and website
* Business contact buttons: Email, Phone, WhatsApp
* Hover-reveal social icons with smooth CSS animations
* "Highlight Member" featured badge
* Responsive 1–4 column grid with gap control
* Color controls for text, background, accent, and hover colors
* Show/hide toggles for bio, social links, and contact info
* Fade-in stagger animation on page load
* Drag-to-reorder members in the editor
* Fully responsive and mobile-friendly
* RTL support and translation ready
* No jQuery — pure vanilla JavaScript
* Optimized performance with lazy-loaded images

**Pro Features (Coming Soon):**

* Carousel / Slider mode
* Popup modal with full member details
* Flip card animation
* Filter members by category
* Custom Post Type and ACF integration
* Advanced animation system (slide, zoom, stagger options)
* "Book a Meeting" CTA button
* AI Bio Generator
* Role-based automatic color theming
* Up to 6 columns

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wbd-team-member` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. In the block editor, search for "WBD Team Member" and add it to your page.
4. Add team members, choose a layout, and customize to your liking.

== Frequently Asked Questions ==

= How many team members can I add? =

There is no limit. You can add as many team members as you need within a single block.

= Can I reorder team members? =

Yes! Use the up/down arrow buttons in the editor to drag and reorder members easily.

= Is it responsive? =

Absolutely. The block is fully responsive and looks great on all devices.

= Does it work with any theme? =

Yes, WBD Team Member is designed to work with any properly coded WordPress theme.
